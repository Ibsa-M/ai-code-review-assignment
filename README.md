This repository contains my submission for the **AI Code Review Assignment**. It includes all tasks, corrected versions, and notes for reference.

## File Structure

- **submission_template.md**  
  Template used for submitting the assignment according to instructions.

- **task1.py, task2.py, task3.py**  
  Original solutions for Task 1, Task 2, and Task 3 of the assignment.

- **correct_task1.py, correct_task2.py, correct_task3.py**  
  Corrected or optimized versions of the tasks after review.

  Notes, explanations, or references used while completing the assignment.

2. Compare with `correct_taskX.py` for corrected or improved implementations.
